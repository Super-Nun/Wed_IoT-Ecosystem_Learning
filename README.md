# Wed_IoT-Ecosystem_Learning

A React-based web application designed to demonstrate an IoT ecosystem by visualizing data fetched from ThingSpeak. This project features a real-time data dashboard using Chart.js to display insights from IoT devices.

## ðŸš€ Features

- **Data Dashboard**: Visualizes data using Bar Charts and Pie Charts.
- **Real-time Updates**: Fetches data from ThingSpeak API at regular intervals.
- **Data Analysis**: Automatically calculates totals and identifies the highest hitting brand/category (e.g., Grab Food, Line Man, Shopee Food).
- **Responsive Design**: Built with Tailwind CSS for a seamless experience on different devices.

## ðŸ“¦ Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

- Node.js installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/Wed_IoT-Ecosystem_Learning.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd Wed_IoT-Ecosystem_Learning
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Application

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173` (or the port shown in your terminal).
